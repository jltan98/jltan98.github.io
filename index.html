About Me

My name is Jia Leng, also known as Cheryl. You may address me by either name.

I am a penultimate student in Singapore Management University studying Information Systems, Business Analytics major.


In the analytical field, I have done projects related to data cleaning and pre-processing, exploratory data analysis, data management, web-scraping (using API calls and python libraries), sentiment analysis, topic modelling, network analysis, use of machine learning models such as Random Forest, Decision Trees, XGBoost etc to train and build classification models, creating dashboard for analysis on Tableau and TIBCO Spotfire, as well as stock analysis (e.g. volatility, returns, risk adjusted return, moving average). Python is used to conduct the above mentioned analysis.


In the software development aspect, I have done projects related to web application development using HTML, CSS, Bootstrap, Javascript, Vue.JS, Python Flask, Docker and MySQL, create high-fi prototype using Axure RP software, did user acceptance testing for the application and CI pipeline using jenkins (connected to GitHub). For deployment/hosting of web app, I have done it using AWS EC2 and RDS.


I am still new to the working industry for Information Technology, hence seeking internship opportunities to improve on my skills in analytics and software development.
